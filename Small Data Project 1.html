# Ultimately, my biggest question is whether the project was a success:
# did I do at least three things more than I didn't?
#
# To figure that out, I calculate the cumulative sum of activities done vs. laziness.
# I felt like it was necessary to reweight my laziness. if I didn't do something during the day, I wasn't 3x
# more lazy than when I wasn't doing something.

activities = np.cumsum(df.active_sum)
activities_diff = np.cumsum(df.active_sum.diff())
laziness = np.cumsum(df.D) / 3
alf = activities/laziness
dlf = activities_diff/laziness

alf.plot()
plt.xlabel('date')
plt.ylabel('activity : laziness ratio')
plt.show()
